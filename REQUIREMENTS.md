# StageSync Requirements

## MVP Phase 1

### Singer Experience
- Quick registration (First, Last, Email).
- Tidal Song Search (restricted to playlist or full catalog toggle).
- Live queue position tracking + Lyrics view.
- Tip links (PayPal, Venmo, CashApp) + Custom message to band.
- **NEW**: View band's social media profiles (FB, IG, TikTok) and website on dashboard.

### Band/Admin Management
- Show Creation: Unique containers for specific dates/venues.
- Queue Control: Set song as played, remove, or reorder.
- Band Profile: Logo, Name, Social Media (FB, IG, TikTok), Website, and Payment link settings.

### SaaS Admin
- Manage Bands: CRUD bands and members.
- User Management: CRUD any user (band member or singer).
- System Analytics.
